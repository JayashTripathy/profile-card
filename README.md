## Getting Started

This is an example of how you can create you own.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/JayashTripathy/profile-card.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run
   ```js
   npm run create 
   ```
#### now fill you all the details and style configurations
